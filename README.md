# 🤖 AI Chatbot using Hugging Face Transformers

## 📌 Project Overview
This project demonstrates the development of an AI-powered chatbot using Hugging Face Transformers. The chatbot is built using the pre-trained DialoGPT model, which is designed for conversational tasks and generates human-like responses.

## 🚀 Features
- Interactive chatbot system
- Generates human-like responses
- Maintains conversation history
- Uses pre-trained NLP model
- Simple and user-friendly implementation

## 🛠️ Technologies Used
- Python
- Hugging Face Transformers
- PyTorch
- Jupyter Notebook / Google Colab

## 🤖 Model Used
- microsoft/DialoGPT-medium  
(A conversational AI model trained for dialogue generation)
